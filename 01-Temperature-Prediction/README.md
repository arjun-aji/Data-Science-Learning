# Temperature Prediction using Linear Regression

## Overview
This project predicts temperature using Linear Regression based on humidity, pressure, latitude, and longitude.

## Files
- Temperature Prediction_Test.ipynb
- humidity.csv

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Results
- Mean Squared Error (MSE): 96.24
- R² Score: 0.504
